Canvas Racers

Canvas Racers is a race game made by 3 elements of the Dead Man Tales Studios.

This game is a single player and localmultplayer game, single player mode consists in the best time you make on the race, basicly like and highscore.

The multiplayer local game consists in trying to be in the 1st place to win while you can pick up and use awesome powerups.
